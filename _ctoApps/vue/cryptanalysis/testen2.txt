The algorithm takes as input a block of 128 bits (16 bytes), the key makes 128, 192 or 256 bits. The 16 input bytes are swapped according to a predefined table. These bytes are then placed in a matrix of 4x4 elements and its lines are rotated to the right. The increment for rotation varies according to the number of the line. A linear transformation is then applied to the matrix, consisting of the binary multiplication of each element of the matrix with polynomials from an auxiliary matrix, this multiplication is subject to special rules according to GF (28) (Galois group or Finished body). The linear transformation ensures better diffusion (bit propagation in the structure) over several turns.
